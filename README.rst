Installation
------------

PyMGRIT requires ``numpy``, ``scipy`` and ``mpi4py``

    >>> pip install pymgrit

or

    >>> python setup.py install

Introduction
------------

PyMGRIT is a library for the Multigrid-Reduction-in-Time (MGRIT) algorithm in Python
